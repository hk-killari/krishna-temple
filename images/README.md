# Images Folder Structure

Please add your temple images with the following names for them to display correctly on the website:

## Main Images (Required)
- `logo.png` - Temple logo (displayed in navbar, recommended: 200x200px)
- `temple-main.jpg` - Main temple image for homepage hero/about section
- `deity.jpg` - Image of Sri Rukmini Satyabhama Sameta Venugopal Swamy
- `temple-architecture.jpg` - Temple architecture/gopuram image
- `og-image.jpg` - Social media sharing image (recommended: 1200x630px)

## Gallery Images (Place in `/images/gallery/` folder)
Create a subfolder called `gallery` and add these images:

### Construction Progress
- `foundation.jpg` - Bhoomi Puja and foundation laying ceremony
- `gopuram.jpg` - Gopuram construction progress
- `main-structure.jpg` - Main temple hall construction
- `stone-carving.jpg` - Artisans working on stone carvings
- `aerial-view.jpg` - Aerial/bird's eye view of temple complex

### Interior Images
- `sanctum.jpg` - Inner sanctum/garbhagriha
- `mandapam.jpg` - Temple mandapam with pillars
- `wall-paintings.jpg` - Traditional murals and paintings
- `ceiling.jpg` - Ceiling art and designs

### Exterior Images
- `entrance.jpg` - Temple main entrance
- `garden.jpg` - Tulsi garden and landscaping
- `night.jpg` - Temple illuminated at night

### Ceremony Images
- `vedic-rituals.jpg` - Havan/yagna ceremony
- `deity-installation.jpg` - Deity installation ceremony
- `yagna.jpg` - Grand yagna ceremony
- `blessing.jpg` - Priests blessing ceremony

## Image Recommendations
- **Format**: JPG or PNG
- **Size**: Optimize for web (under 500KB per image)
- **Aspect Ratio**: 
  - Gallery images: 4:3 or 16:9
  - Logo: Square (1:1)
  - OG Image: 1200x630px
- **Quality**: High resolution but compressed for web

## Fallback Behavior
If any image is missing, the website will automatically display a placeholder emoji icon instead.
